# Reflection

This capstone demonstrates how AI agents can enhance enterprise expense management by assisting—not replacing—human decision-makers. The architecture emphasizes modularity, transparency, auditability, and security while remaining extensible for future integrations through MCP and cloud-native deployment.

The project reflects both the technical concepts taught during the course and practical software engineering principles suitable for real-world enterprise environments.
