# 5 Minute Demo Script

## Introduction
Hello everyone. Today I am presenting Ambience ExpenseFlow, an enterprise AI agent platform for travel and expense management.

## Problem
Expense management is often manual, slow, and difficult to audit.

## Solution
Ambience ExpenseFlow uses agents to automate expense submission, compliance review, manager approval, document tracking, and audit timelines.

## Demo Flow
1. Show login page.
2. Show My Reports.
3. Show Submit Expense.
4. Show Supporting Documents.
5. Show Pending Approvals.
6. Show Expense History.
7. Show Audit Timeline.

## Technologies
Google ADK, Cloud Run, Firestore, Pub/Sub, Vertex AI, Google OAuth, and Antigravity.

## Closing
This project demonstrates how AI agents can improve enterprise compliance, efficiency, and transparency.
